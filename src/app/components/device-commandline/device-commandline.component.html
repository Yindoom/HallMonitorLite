Commands will be executed in /bin
Need Commandline help?

Remember to add the path where you want to run the command if necessary, fx mkdir /home/pi/hallmonitor_lite/hello_world.
Usefull command sequences:

For rebooting:
sudo reboot

For starting the ssh tunnel service from the Raspberry Pi to the backend:
sudo systemctl enable autossh-rpi-tunnel.service
sudo systemctl start autossh-rpi-tunnel.service

Remember to stop the the ssh tunnel service from the Raspberry Pi to the backend:
sudo systemctl stop autossh-rpi-tunnel.service
sudo systemctl disable autossh-rpi-tunnel.service

For connecting trough the tunnel on the backend type:
ssh pi@localhost -p 9001
9001 is the port used by device 1.

For updating the code:
cd /home/pi/hallmonitor_lite/ && git pull
